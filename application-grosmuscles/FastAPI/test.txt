# ============================================================
# 🚀 FASTAPI BACKEND ET FRONTEND — GUIDE DE DÉMARRAGE
# ============================================================
# 1. BACKEND (FastAPI)
# --------------------
# Ouvre un terminal dans le dossier `FastAPI`.
#
# a. Installer les dépendances (la première fois) :
#    pip install -r requirement.txt
#
# b. Lancer le serveur :
#    uvicorn main:app --reload --host 0.0.0.0 --port 8000
#
# 2. FRONTEND (Expo)
# ------------------
# Ouvre un SECOND terminal dans le dossier `frontend`.
#
# a. Installer les dépendances (la première fois) :
#    npm install
#
# b. Lancer l'application :
#    npx expo start
#
# 3. NOTES IMPORTANTES
# --------------------
# - Le backend doit être lancé pour que l'application fonctionne.
# - La détection de l'IP du backend est automatique dans `frontend/services/api.js`.
# - Pour un appareil physique, le mode tunnel est recommandé si le QR code ne marche pas :
#   npx expo start --tunnel
# - La base de données SQLite se trouve dans `FastAPI/grosmuscles.db`.
# ============================================================
